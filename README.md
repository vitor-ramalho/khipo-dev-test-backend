<h1>Developer test MB-DATA Backend</h1>

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

Use only Pinterest images in requests [https://br.pinterest.com/](https://br.pinterest.com/)
